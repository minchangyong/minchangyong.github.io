# qts.getAppTrackInfo()

以 Promise 风格 调用：

支持

# 功能描述

# 获取客户端当前页面埋点数据

# success 回调函数

属性 类型 说明
businessType String 埋点业务类型
currentId String 客户端当前页面地址
deviceId String 客户端设备号
jwtToken String 服务端登录态
eventType Number 埋点类型，1曝光，2点击，5页面打开，7心跳
latitude Number 纬度，范围为 -90~90，负数表示南纬
longitude Number 经度，范围为 -180~180，负数表示西经
positionId String 点位ID
positionIdFir String 一级页面ID
positionIdSec String 二级页面ID
positionIdThi String 三级页面ID
prePage String 上个页面ID，同referId
preSourceId String 一二三级ID集合
referId String 客户端上个页面地址
source Boolean 是否开启溯源
sourceId String 溯源一二三级ID集合
startPositionExt Object 埋点额外信息，如remark等
timestamp Number 时间戳
townId Number 当前定位城市ID
trackingSource Boolean 是否开启顶级溯源
version String 客户端版本号
