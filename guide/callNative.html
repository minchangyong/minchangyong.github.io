# qts.postNotificationCenter()

以 Promise 风格 调用：

支持

# 功能描述

# h5做了以下类型操作时，需要通知给客户端。

# 请求参数

属性 类型 说明
eventName String 方法名
eventParams Object 方法所需的参数

# eventName枚举

属性 类型 说明 参数
login String 登录成功 --
logout String 退出登录 --
partJobShareSuccess String 兼职分享成功 eventParams : { partJobId: 1000 }
apply_partJob_success_notification String 兼职报名成功 eventParams : { partJobId: 1000 }
easyTaskShareSuccess String 小任务分享成功 eventParams : { taskBaseId: 1000 }
easyTaskApplySuccess String 小任务领取成功 eventParams : { taskBaseId: 1000 }
userResumeInfoDidChange String 用户简历信息发生变化 --
realNameAuthSuccess String 实名认证成功 --

# 示例代码

// 报名了岗位id为3022764时，通知客户端
qts.postNotificationCenter('apply_partJob_success_notification', { partJobId: 3022764 }).then((res) => {
	console.log(res)
})
