    # 简介

    # 关于QTS-JSSDK

    # QTS-JSSDK 提供丰富的青团社原生 API，可以方便调用青团社客户端提供的能力，达到媲美原生应用的体验，如页面跳转，分享功能等。
    # 1.1 安装使用
    // 首先在对应的项目安装私有npm包
npm install @qts/jsbridge
    # 1.2 快速开始
    // 然后在用到了jsBridge的地方引用：
import qts from '@qts/jsbridge'

// 使用时直接调用即可，统一使用promise的形式，部分API使用callback形式，like: qts.onPageEvent(res => {})。
qts.displayAppHead()

// 获取客户端、用户信息
qts.getAppInfo().then((res) => {
  console.log(res)
})
